# Web BDF/uInt array PIXEL FONT EDITOR
<p align="center">
  <a href="./.github/ss01.png"><img src="./.github/ss01.png" alt="App UI" /></a>
</p>
This app is a BDF font/uInt array pixel font editor/creator contained within a single .html file, using vanilla JavaScript. 
 
It can be ran locally offline using any modern browser.
 
The only external dependency are Google Fonts used in the UI, but the app will still run fine wothout them.

**Disclaimer: This app was made with the help of Claude AI.**<br>
For this reason I was reluctant to make the repo public, but I’ve put many hours into it myself and made multiple manual UI and JS tweaks to make sure it can look and work how I envisioned it. Hopefully someone will find it useful.

## Features
* Import and export BDF fonts
* Import and export .h files (uInt array)
* Create font from scratch
* Set glyph size and offset
* Modify existing or Create new glyphs
* Clone, name, remove glyphs
* Grid helpers: Center mark and helper lines
* Copy currently edited glyph as BDF/uInt array 
* Text preview
* Separate zoom for glyph editor and text preview
* Save/Autosave editor state using browser's localStorage
